Welcome to the Past.

Obviously that is loooong before we even thought about anything remotely to React.

This is how the first webpages looked like. No styling or no interactions. Just some plain old text at maximum with some font styles.

Their functions were pretty clear. Document information. A collection of paragraphs. Let's run through a fast and simple timeline:

  • April 1999: The term Web 2.0 was coined. It mainly defines the internet as a place for content created by the users.
    This includes social media, forums, encyclopedias and so on.
    We also reached the point of a flashy and colorful time, websites stared to begin being interactive!
  • 2006:b A JavaScript framework called "jQuery" comes into place. It emphasized the interaction between JavaScript and HTML.
  • May 2013: React was released. "The open-source React library is designed to make it easier to create user interfaces and UI components for web applications. React can also be used to create single-page or mobile applications."
